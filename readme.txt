== Description ==

This plugin routes your emails to MailHog for development with Docker.

== Installation ==

1. Upload the entire `lab-mailhog` folder to your `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

No questions. Just activate and enjoy.
